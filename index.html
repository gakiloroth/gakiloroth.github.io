Servants Manager Quests Manager Battle Simulator How to Use

Select a servant to load attack and passive values. Use the checkboxes to change the attack match the servant's max grailed or max fou/gold fou'd value. Then, you can load Craft Essence data with the "Choose Craft Essence" and "Add CE Stats", or you can add those values manually.
Servant NP upgrades are assumed and up to date with JP - you may have to manually adjust the NP Damage % accordingly if on NA or if the servant has not done their NP upgrade.
Hover over the for more details about certain fields, and use the README if you want further details.
